# Análisis del caso
Lo que sucedió fue que un desarrollador filtró por error los datos de una API de ChatGpt, por lo que varios bots automatizados, detectaron este error, y la comenzaron a usar, generándole costos de facuración excesivos, se hubiera evitado, agregando un .env al .gitignore, pues así no tendrían acceso a la API KEY ni a los datos confidenciales.
